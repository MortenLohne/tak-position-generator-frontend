Random tak position generator

What is this?

This page implements a bijection between tak positions and natural numbers, meaning that every number between 0 and 234953877228339135776421063941057364108851372312359712 maps to a unique tak position. This also allows generating truly random tak positions, by generating a random number in that range and converting it to a position.

The math is based on the counting function described in AI agents for the abstract strategy game Tak by Laurens Beljaards, section 4.2.
